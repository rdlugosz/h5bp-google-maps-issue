Demonstration of Google Maps printing issue in H5BP.

  • A map that is too WIDE (set via px > ~800 or via 100% and a wide browser) will look fine on screen, but if you ctrl-p to print you'll note that the map tiles are BLANK. (You should be able to try this right now with this page!)
  • Overriding the print media query that sets max-width: 100% !important; resolves the issue.
  • Please see the HEAD section of this file for a working demo; by uncommenting the print media query override you should be able to print the map. You can even just change it via dev tools and you'll see the results (immediately if you have print Emulation enabled!)
  • Tested on Chrome 46.0.2490.52 beta, but witnessed on other webkit browsers.
